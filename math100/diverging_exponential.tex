\begin{theorem}
The integral
$$\int_1^\infty \frac{e^x}{x} ~dx$$
diverges.
\end{theorem}

We don't know how to compute the antiderivative of $e^x/x$. (In fact it is a function called $\mathrm{Ei}(x)$ which we haven't studied.)
So, to prove this, we should use direct comparison.
If we use desmos or a graphing calculator to plot this function, it looks like for $x \geq e$ it's always greater than $e$.
This suggests that we should use the comparison $e^x/x \geq e$.
 
\begin{proof}
We know that $e^1/1 = e \geq e$.
Also, we know that
$$\frac{d}{dx} \left[\frac{e^x}{x}\right] = \frac{e^x(x - 1)}{x^2}$$
which is positive if $x > 1$.
So $e^x/x$ is increasing on the interval $[1, \infty)$, so $e^x/x \geq e$ for every $x \geq 1$.

But we also know that
$$\int_1^\infty e ~dx = \lim_{R \to \infty} \int_1^R e ~dx = \lim_{R \to \infty} e(R - 1) = +\infty$$
so $\int_1^\infty e~dx$ diverges.
So, by direct comparison, $\int_1^\infty e^x/x ~dx$ diverges.
\end{proof}
