Course description: The official course description is Analytic functions of a complex variable. Cauchy's integral theorem, power series, Laurent series, singularities of analytic functions, the residue theorem with application to definite integrals. Some additional topics such as conformal mapping. This course can be thought of as a "sequel" to Math 53 in the following sense: in Math 53, you considered line integrals of functions R2 → R2. These can be also thought of us as functions C → C. If a function C → C satisfies a very innocent-looking condition called "holomorphy", then it automatically solves a certain differential equation, automatically equals its own Taylor series, is automatically conservative, and much more. For example, to compute the value of the function at a point, one just needs to find an average in a ball around that point. Miraculously, the functions you learned about in high school R → R extend to holomorphic functions, and so we can use complex analysis to solve problems in single-variable calculus.
