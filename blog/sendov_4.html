Lemma 1 Let $latex {K \subseteq \mathbf C}&fg=000000$ be a compact set. Then

$latex \displaystyle \mathbf E \log |z - \zeta| \leq \log |z| + o(1)&fg=000000$

$latex \displaystyle -\frac{\log n}{n - 1} - \frac{1}{n - 1} \log |f'(z)| = U_\zeta(z) = -\mathbf E \log |z - \zeta|&fg=000000$

5. Fine control

We finish this series. If you want, you can let $latex {\varepsilon > 0}&fg=000000$ be a standard real. I think, however, that it will be easier to think of $latex {\varepsilon}&fg=000000$ as ``infinitesimal, but not as infinitesimal as the term of the form o(1)". In other words, $latex {1/n}&fg=000000$ is smaller than any positive element of the ordered field $latex {\mathbf R(\varepsilon)}&fg=000000$; briefly, $latex {1/n}&fg=000000$ is infinitesimal with respect to $latex {\mathbf R(\varepsilon)}&fg=000000$. We still reserve $latex {o(1)}&fg=000000$ to mean an infinitesimal with respect to $latex {\mathbf R(\varepsilon)}&fg=000000$. Now $latex {\varepsilon^n = o(1)}&fg=000000$ by underspill, since this is already true if $latex {\varepsilon}&fg=000000$ is standard and $latex {0 < \varepsilon < 1}&fg=000000$. Underspill can also be used to transfer facts at scale $latex {\varepsilon}&fg=000000$ to scale $latex {1/n}&fg=000000$. I think you can formalize this notion of ``iterated infinitesimals" by taking an iterated ultrapower of $latex {\mathbf R}&fg=000000$ in the theory of ordered rings.

Let us first bound $latex {\log |a|}&fg=000000$. Recall that $latex {|a| \leq 1}&fg=000000$ so $latex {\log |a| \leq 0}&fg=000000$ but in fact we can get a sharper bound. This will give us bounds on $latex {1 - a}&fg=000000$ when we take the Taylor expansion

$latex \displaystyle \log|a| = -(1 - a)(1 + o(1)).&fg=000000$

Lemma 9 One has

Theorem 11 (Jensen's formula) Let $latex {g: D(0, 1) \rightarrow \mathbf C}&fg=000000$ be a holomorphic function with zeroes $latex {a_1, \dots, a_n \in D(0, 1)}&fg=000000$ and $latex {g(0) \neq 0}&fg=000000$. Then

$latex \displaystyle \log |g(0)| = \sum_{j=1}^n \log |a_j| + \frac{1}{2\pi} \int_0^{2\pi} \log |g(e^{i\theta})| ~d\theta.&fg=000000$

In hindsight this is kinda trivial but I never realized it. In fact $latex {\log |g|}&fg=000000$ is subharmonic and in fact its Laplacian is exactly a linear combination of delta functions at each of the zeroes of $latex {g}&fg=000000$. If you subtract those away then this is just the mean-value property

$latex \displaystyle \log |g(0)| = \frac{1}{2\pi} \int_0^{2\pi} \log |g(e^{i\theta})| ~d\theta.&fg=000000$

Lemma 12 For any standard $latex {\varphi \in C^\infty(\partial D(0, 1))}&fg=000000$,

$latex \displaystyle \int_0^{2\pi} \varphi(e^{i\theta}) U_\zeta(e^{i\theta}) ~d\theta = o(\sigma^2) + o(1)^n.&fg=000000$

Besides,

$latex \displaystyle U_\zeta(a) = o(\sigma^2) + o(1)^n.&fg=000000$

Proof: Let $latex {m}&fg=000000$ be the Haar measure on $latex {\partial D(0, 1)}&fg=000000$. We first prove this when $latex {\varphi \geq 0}&fg=000000$. Since $latex {T}&fg=000000$ is discrete and $latex {\partial D(0, 1)}&fg=000000$ is compact, for any standard (or semi-infinitesimal) $latex {\delta > 0}&fg=000000$, there is a standard compact set

$latex \displaystyle I \subseteq \partial D(0, 1) \setminus S&fg=000000$

such that

$latex \displaystyle m(\partial D(0, 1) \setminus I) < \delta.&fg=000000$

By the previous lemma, if $latex {e^{i\theta} \in I}&fg=000000$ then

$latex \displaystyle \varphi(e^{i\theta}) U_\zeta(a) - \varphi(e^{i\theta}) U_\zeta(e^{i\theta}) \geq -o(\sigma^2) - o(1)^n&fg=000000$

and the same holds when we average in Haar measure:

$latex \displaystyle U_\zeta(a)\int_I \varphi~dm - \int_I \varphi(e^{i\theta}) U_\zeta(e^{i\theta})~dm(e^{i\theta}) \geq -o(\sigma^2) - o(1)^n.&fg=000000$

We have

$latex \displaystyle |\log |e^{i\theta} - \zeta| + \text{Re } e^{-i\theta}\zeta| \leq |\log|3 - \zeta| + 3\text{Re } \zeta| \in L^2(dm(e^{i\theta}))&fg=000000$

so, using the Cauchy-Schwarz inequality, one has

$latex \displaystyle \int_{\partial D(0, 1) \setminus I} \varphi(e^{i\theta}) (\log |e^{i\theta} - \zeta| + \text{Re } e^{-i\theta}\zeta) ~dm(e^{i\theta}) = \sqrt{\int_I |\log |e^{i\theta} - \zeta| + \text{Re } e^{-i\theta}\zeta|} = O(\delta^{1/2}).&fg=000000$

Meanwhile, if $latex {|\zeta| \leq 1/2}&fg=000000$ then the fact that

$latex \displaystyle \log |e^{i\theta} - \zeta| = \text{Re }-\frac{\zeta}{e^{i\theta}} + O(|\zeta|^2)&fg=000000$

implies

$latex \displaystyle \log |e^{i\theta} - \zeta| + \text{Re } \frac{\zeta}{e^{i\theta}} = O(|\zeta|^2)&fg=000000$

and hence

$latex \displaystyle \int_{\partial D(0, 1) \setminus I} \varphi(e^{i\theta}) (\log |e^{i\theta} - \zeta| + \text{Re } e^{-i\theta}\zeta) ~dm(e^{i\theta}) = O(\delta|\zeta|^2).&fg=000000$

We combine these into the unified estimate

$latex \displaystyle \int_{\partial D(0, 1) \setminus I} \varphi(e^{i\theta}) (\log |e^{i\theta} - \zeta| + \text{Re } e^{-i\theta}\zeta) ~dm(e^{i\theta}) = O(\delta^{1/2}|\zeta|^2)&fg=000000$

valid for all $latex {|\zeta| \leq 1}&fg=000000$, hence almost surely. Taking expected values we get

$latex \displaystyle \int_{\partial D(0, 1) \setminus I} \varphi(e^{i\theta})U_\zeta(e^{i\theta}) + \varphi(e^{i\theta}) \text{Re }e^{-i\theta}\mu ~dm(e^{i\theta}) = O(\delta^{1/2}(|\mu|^2 + \sigma^2)) + o(\sigma^2) + o(1)^n.&fg=000000$

In the last lemma we bounded $latex {|\mu|}&fg=000000$ so we can absorb all the terms with $latex {\mu}&fg=000000$ in them to get

$latex \displaystyle \int_{\partial D(0, 1) \setminus I} \varphi(e^{i\theta})U_\zeta(e^{i\theta}) ~dm(e^{i\theta}) = O(\delta^{1/2}\sigma^2) + o(\sigma^2) + o(1)^n.&fg=000000$

We also have

$latex \displaystyle \int_{\partial D(0, 1) \setminus I} \varphi ~dm = O(\delta)&fg=000000$

(here Tao refers to a mysterious undefined measure $latex {\sigma}&fg=000000$ but I'm pretty sure he means $latex {m}&fg=000000$). Putting these integrals together with the integrals over $latex {I}&fg=000000$,

$latex \displaystyle \ U_\zeta(a)int_{\partial D(0, 1)} \varphi ~dm - \int_{\partial D(0, 1)} \varphi(e^{i\theta}) U_\zeta(e^{i\theta}) ~dm(e^{i\theta}) \geq -O(\delta^{1/2}\sigma^2) - o(\sigma^2) - o(1)^n.&fg=000000$

By underspill we can delete $latex {\delta}&fg=000000$, thus

$latex \displaystyle U_\zeta(a)\int_{\partial D(0, 1)} \varphi ~dm - \int_{\partial D(0, 1)} \varphi(e^{i\theta}) U_\zeta(e^{i\theta}) ~dm(e^{i\theta}) \geq - o(\sigma^2) - o(1)^n.&fg=000000$

We now consider the specific case $latex {\varphi = 1}&fg=000000$. Then

$latex \displaystyle U_\zeta(a) - \int_{\partial D(0, 1)} U_\zeta ~dm \geq -o(\sigma^2) - o(1)^n.&fg=000000$

Now Tao claims and doesn't prove

$latex \displaystyle \int_{\partial D(0, 1)} U_\zeta ~dm = 0.&fg=000000$

To see this, we expand as

$latex \displaystyle \int_{\partial D(0, 1)} U_\zeta ~dm = -\mathbf E \frac{1}{2\pi} \int_0^{2\pi} \log|\zeta - e^{i\theta}| ~d\theta&fg=000000$

using Fubini's theorem. Now we use Jensen's formula with $latex {g(z) = \zeta - z}&fg=000000$, which has a zero exactly at $latex {\zeta}&fg=000000$. This seems problematic if $latex {\zeta = 0}&fg=000000$, but we can condition on $latex {|\zeta| > 0}&fg=000000$. Indeed, if $latex {\zeta = 0}&fg=000000$ then we have

$latex \displaystyle \int_0^{2\pi} \log|\zeta - e^{i\theta}| ~d\theta = \int_0^{2\pi} \log 1 ~d\theta = 0&fg=000000$

which already gives us what we want. Anyways, if $latex {|\zeta| > 0}&fg=000000$, then by Jensen's formula,

So that's how it is. Thus we have

$latex \displaystyle -U_\zeta(a) \leq o(\sigma^2) + o(1)^n.&fg=000000$

Since $latex {|a - \zeta| \geq 1}&fg=000000$, $latex {\log |a - \zeta| \geq 0}&fg=000000$, so the same is true of its expected value $latex {-U_\zeta(a)}&fg=000000$. This gives the desired bound

We can use that bound to discard $latex {U_\zeta(a)}&fg=000000$ from the average

$latex \displaystyle U_\zeta(a)\int_{\partial D(0, 1)} \varphi ~dm - \int_{\partial D(0, 1)} \varphi(e^{i\theta}) U_\zeta(e^{i\theta}) ~dm(e^{i\theta}) \geq - o(\sigma^2) - o(1)^n,&fg=000000$

$latex \displaystyle \int_{\partial D(0, 1)} \varphi(e^{i\theta}) U_\zeta(e^{i\theta}) ~dm(e^{i\theta})= o(\sigma^2) + o(1)^n.&fg=000000$
