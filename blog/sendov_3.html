2. Convergence in $latex {L^1_{loc}}&fg=000000$

$latex \displaystyle \int_K g ~d(\nu^{(\infty)} - \nu) = o(1).&fg=000000$

$latex \displaystyle s_{\lambda^{(\infty)}}(z) - s_\lambda(z) = \int_{D(0, 1)} \frac{d\nu^{(\infty)}(w)}{z - w} - \int_{D(0, 1)} \frac{d\nu(w)}{z - w}.&fg=000000$

$latex \displaystyle \int_K \int_{D(0, 1)} \frac{d(\nu^{(\infty)} - \nu)(w)}{|z - w|} ~d\rho(z) = \int_{D(0, 1)} \int_K \frac{d\rho(z)}{|z - w|} d(\nu^{(\infty)} - \nu)(w)&fg=000000$

$latex \displaystyle \int_K \int_{D(0, 1)} \frac{d(\nu^{(\infty)} - \nu)(w)}{|z - w|} ~d\rho(z) = o(1)&fg=000000$

which gives what we want when we recall

$latex \displaystyle s_\lambda(z) = \frac{1}{n} \frac{f'(z)}{f(z)}&fg=000000$

and we plug in $latex {s_\lambda}&fg=000000$.

3. Case one: Outlining the proof

The proof for case one is much longer, and is motivated by the pseudo-counterexample

$latex \displaystyle f(z) = z^n - 1.&fg=000000$

Here $latex {a}&fg=000000$ is an $latex {n}&fg=000000$th root of unity, and $latex {f}&fg=000000$ has no critical points on $latex {D(a, 1)}&fg=000000$, but does have $latex {n - 1}&fg=000000$ critical points at $latex {0 \in \partial D(a, 1)}&fg=000000$. Similar pseudo-counterexamples hold for

$latex \displaystyle 1 - o(1) \leq a \leq 1 - \varepsilon_0^n&fg=000000$

where $latex {\varepsilon_0 > 0}&fg=000000$ is standard. We will seek to control these examples by controlling $latex {\zeta}&fg=000000$ up to an error of size $latex {O(\sigma^2) + o(1)^n}&fg=000000$; here $latex {\sigma^2}&fg=000000$ is the variance of $latex {\zeta}&fg=000000$ and $latex {o(1)^n}&fg=000000$ is an infinitesimal raised to the power of $latex {n}&fg=000000$, thus is very small, and forces us to balance out everything in terms of $latex {a}&fg=000000$.

As discussed in the introduction of this post, $latex {\zeta}&fg=000000$ is infinitesimal in probability (and, in particular, its expected value $latex {\mu}&fg=000000$ is infinitesimal); thus, with overwhelming probability, the critical points of $latex {f}&fg=000000$ are all infinitesimals. Combining this with the fact that $latex {\lambda^{(\infty)}}&fg=000000$ is uniformly distributed on $latex {\partial D(0, 1)}&fg=000000$, it follows that $latex {f}&fg=000000$ sort of looks like $latex {f(z) = z^n - 1}&fg=000000$.

We start with some nice bounds:

Lemma 6 (preliminary bounds) For any standard compact set $latex {K \subset \mathbf C}&fg=000000$, one has

$latex \displaystyle f(z) = f(0) + O((|z| + o(1))^n)&fg=000000$

and

$latex \displaystyle f'(z) = O((|z| + o(1))^n)&fg=000000$

uniformly in $latex {z \in K}&fg=000000$.

In other words, $latex {f}&fg=000000$ sort of grows like the translate of a homogeneous polynomial of degree $latex {n}&fg=000000$.

It would be nice if $latex {\zeta}&fg=000000$ was infinitesimal in $latex {L^\infty}&fg=000000$, but this isn't quite true; the following lemma is the best we can do.

Lemma 7 (uniform convergence of $latex {\zeta}&fg=000000$) There is a standard compact set

$latex \displaystyle S = (\overline{D(0, 1)} \cap \partial D(1, 1)) \cup T&fg=000000$

where $latex {T}&fg=000000$ is countable, standard, does not meet $latex {\overline{D(1, 1)}}&fg=000000$, and consists of isolated points of $latex {S}&fg=000000$, such that $latex {d(\zeta, S)}&fg=000000$ is infinitesimal in $latex {L^\infty}&fg=000000$.

So we think of $latex {S}&fg=000000$ as some sort of generalization of $latex {0}&fg=000000$. Away from $latex {S}&fg=000000$ we have good bounds on $latex {f,f'}&fg=000000$:

Lemma 8 (approximating $latex {f,f'}&fg=000000$ outside $latex {S}&fg=000000$) For any standard compact set $latex {K \subset \mathbf C \setminus S}&fg=000000$:
  1. Uniformly in $latex {z, w \in K}&fg=000000$,

    $latex \displaystyle f'(w) = (1 + O(n|z - w|\sigma^2|e^{o(n|z -w|)})) \frac{f'(z)}{(z - \mu)^{n-1}} (w - \mu)^{n-1}.&fg=000000$

    $latex \displaystyle f(z) = f(0) + \frac{1 + O(\sigma^2)}{n} f'(z) (z - \mu) + O((\varepsilon + o(1))^n).&fg=000000$

As a consequence, we can show that every zero of $latex {f}&fg=000000$ which is far from $latex {S}&fg=000000$ is close to the level set

$latex \displaystyle U_\zeta = \frac{1}{n} \log\frac{1}{|f(0)|}.&fg=000000$

This in particular holds for $latex {a}&fg=000000$, since the standard part of $latex {a}&fg=000000$ is $latex {1}&fg=000000$, and $latex {T}&fg=000000$ does not come close to $latex {1}&fg=000000$ (so neither does $latex {S}&fg=000000$). In fact the error term is infinitesimal:
Lemma 9 (zeroes away from $latex {S}&fg=000000$) For any standard compact set $latex {K \subset \mathbf C \setminus S}&fg=000000$, any standard $latex {\varepsilon > 0}&fg=000000$, and any zero $latex {\lambda_0 \in K}&fg=000000$,

$latex \displaystyle U_\zeta(\lambda_0) = \frac{1}{n} \log \frac{1}{|f(0)|} + O(n^{-1}\sigma^2) + O((\varepsilon + o(1))^n)&fg=000000$

uniformly in $latex {\lambda_0}&fg=000000$.

$latex \displaystyle U_\zeta(\lambda) - U_\zeta(a) = O(n^{-1}\sigma^2 + (\varepsilon + o(1))^n)&fg=000000$

is infinitesimal. This gives us some more bounds:
Lemma 10 (fine control) For every standard $latex {\varepsilon > 0}&fg=000000$:
  1. One has

    $latex \displaystyle \mu, 1 - a = O(\sigma^2 + (\varepsilon + o(1))^n).&fg=000000$

  2. For every compact set $latex {I \subseteq \partial D(0, 1) \setminus S}&fg=000000$ and $latex {e^{i\theta} \in I}&fg=000000$,

    $latex \displaystyle U_\zeta(a) - U_\zeta(e^{i\theta}) -o(\sigma^2) - o(1)^n.&fg=000000$

  3. For every standard smooth function $latex {\varphi: \partial D(0, 1) \rightarrow \mathbf C}&fg=000000$,

    $latex \displaystyle \int_0^{2\pi} \varphi(e^{i\theta}) U_\zeta(e^{i\theta}) ~d\theta = o(\sigma^2) + o(1)^n.&fg=000000$

    $latex \displaystyle U_\zeta(a) = o(\sigma)^2 + o(1)^n.&fg=000000$

Here Tao claims

$latex \displaystyle \int_0^{2\pi} e^{-2i\theta} \log \frac{1}{|e^{i\theta} - \zeta|} ~d\theta = \frac{\pi}{2}\zeta^2.&fg=000000$

Apparently this follows from Fourier inversion but I don't see it. In any case if we take the expected value of the left-hand side we get

$latex \displaystyle \int_0^{2\pi} \mathbf Ee^{-2i\theta} \log \frac{1}{|e^{i\theta} - \zeta|} ~d\theta = \int_0^{2\pi} e^{-2i\theta} U_\zeta(e^{i\theta}) = o(\sigma^2) + o(1)^n&fg=000000$

by the fine control lemma, so

$latex \displaystyle \mathbf E \zeta^2 = o(\sigma^2) + o(1)^n.&fg=000000$

In particular this holds for the real part of $latex {\zeta}&fg=000000$. Since $latex {\sigma^2}&fg=000000$ is infinitesimal, so are the first two moments of the real part of $latex {\zeta}&fg=000000$.

Since $latex {|a - \zeta| \in [1, 2]}&fg=000000$, one has

$latex \displaystyle |a - \zeta| - 1 \sim \log |a - \zeta|.&fg=000000$

This is true since for any $latex {s \in [1, 2]}&fg=000000$ one has $latex {\log s \sim s - 1}&fg=000000$ (which follows by Taylor expansion). In particular,

$latex \displaystyle |1 - \zeta| \leq |1 - a| + |a - \zeta| = 1 + O((1 - a) + \log |a - \zeta|).&fg=000000$

Let $latex {\tilde \zeta}&fg=000000$ be the best approximation of $latex {\zeta}&fg=000000$ on the arc $latex {\partial D(1, 1) \cap \overline{D(0, 1)}}&fg=000000$, which exists since that arc is compact; then

$latex \displaystyle |\zeta - \tilde \zeta| = O((1 - a) + \log|a - \zeta|).&fg=000000$

Since $latex {\tilde \zeta \in \partial D(1, 1)}&fg=000000$, it has the useful property that

$latex \displaystyle \text{arg }\zeta \in [\pi/3,\pi/2] \cup [-\pi/2, -\pi/3];&fg=000000$

therefore

$latex \displaystyle \text{Re } \tilde \zeta^2 \leq -\frac{1}{2} |\tilde \zeta|^2.&fg=000000$

Plugging in the expansion for $latex {\tilde \zeta}&fg=000000$ we have

$latex \displaystyle \text{Re } \zeta^2 \leq -\frac{1}{2} |\zeta|^2 + O(|\zeta|((1-a) + \log|a -\zeta|) + ((1 - a) + \log|a - \zeta|)^2).&fg=000000$

We now use the inequality $latex {2|zw| \leq |z|^2 + |w|^2}&fg=000000$ several times. First we bound

$latex \displaystyle \frac{1}{2} |\zeta|((1-a) + \log|a -\zeta|) \leq \frac{1}{4}|\zeta|^2 + O(\log^2 |a - \zeta|).&fg=000000$

I had to think a bit about why this is legal; the point is that you can absorb the implied constant on $latex {\zeta}&fg=000000$ into the implied constant on $latex {\log |a - \zeta|}&fg=000000$ before applying the inequality. Now we bound

$latex \displaystyle ((1 - a) + \log|a - \zeta|)^2 = (1 - a)^2 + 2(1 - a)\log|a - \zeta| + \log^2 |a - \zeta| = O((1-a)^2 + \log^2 |a - \zeta|)&fg=000000$
